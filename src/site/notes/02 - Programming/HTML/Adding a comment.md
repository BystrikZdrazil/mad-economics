# How to add a comment to HTML code
Code is added to HTML for better readability and understanding by humans working on that file. These comments are undetectable by the browser.
- Syntax:
	- ```<!--TEXT-->```

```html
<!--Shows as a comment-->
<h1>
	Title example
</h1>
```
