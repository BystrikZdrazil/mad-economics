# Expatriation

| **Pros**                                  | **Cons**                               |
| ----------------------------------------- | -------------------------------------- |
| Employee follows your culture             | May know less about the target culture |
| Easier communication with the employee    | In some cases may be more expensive    |
| In some cases may be cheaper              |                                        |
| No cultural clashes and misunderstandings |                                        |
| Knows the product/service better          |                                        |

# Hiring local

| **Pros**                         | **Cons**                                     |
| -------------------------------- | -------------------------------------------- |
| Knows a lot about target culture | May be difficult to communicate with         |
| In some cases may be cheaper     | May not want to adapt to company culture     |
|                                  | May not be loyal to a foreign company        |
|                                  | May not know all the product/service details |
