# Boundaries between people and the group

| **Individual is central** | **Group is central**    |
| ------------------------- | ----------------------- |
| Individualism (Hoftede)   | Collectivism (Hofstede) |
| Autonomy (Schwartz)       | Embededdness (Schwartz) |

# Equality or inequality between people

| **Inequality**      | **Equality**       |
| ------------------- | ------------------ |
| High power distance | Low power distance |
| Hierarchy           | Egalitarian        |

# Acting with or for others

| **Assrtive: with others** | **Nurturing: for others** |

# Dealing with uncertainty

| **Uncertainty is bad** | **Uncertainty is inevitable** |
| ---------------------- | ----------------------------- |
| Strong UA (Hofstede)   | Weak UA (Hofstede)            |
