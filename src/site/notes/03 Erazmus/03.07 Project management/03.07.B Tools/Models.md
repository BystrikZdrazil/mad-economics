# DESC Model
- Describe, express, specify, consequences
    - Describe - explain the situation
    - Express - what impact the described situation had had on you
    - Specify
    - Consequences

# CEDAR Model
- Context, examples, diagnoses, actions, review
    - Context - paint the scene & explain context
    - Examples - give clear examples of the behaviour
    - Diagnosis - diagnose the recipient and ask for reasons
    - Actions - focus on future actions and reflect with the recipient
    - Review - discuss and conclude

# 360° Model
- Collect feedback on someone’s performance or actions from as many sources as possible
- Self-evaluation should be included
- Neutral analysis by a third party
- According to the feedback future actions can be planned
