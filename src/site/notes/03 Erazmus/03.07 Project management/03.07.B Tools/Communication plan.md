# The importance of communication
- Aligns goals and expectations
- Keep stakeholders informed
- Resource coordination
- Risk identification
- Team cohesion and accountability

# 5 core components
1. **Sender**
	- Initiates the communicaton
	- Often the project manager
2. **Recipients**
	- Receive the communication
	- Can be teams or individuals
3. **Description**
	- The communication itself
	- Certain information and details
4. **Method**
	- The media used to communicate
5. **Frequency**
	- Daily, twice a week, weekly, monthly,...

# Communication matrix

|Sender|Recipient|Description|Method|Frequency|
|---|---|---|---|---|
|Project Manager|Core Project Team|Weekly Status Report|Email|Fridays at 9 am|
|Project Manager|Core Project Team|Weekly Project Team Meeting|Meeting|Wednesday at 11 am|
|Project Manager|Core Project Team|Meeting Minutes|SharePoint|As needed|
|Marketing Lead|Customers|Product Launch Notice|Email|2/17/2023|

# How to make it work

## Project planning phase
- Outine the specific tasks and expectations
- Delegate one team member to communicate
- Specify required frequency
- Choose a storage for all communication

## Execution and control phase
- Tailor it to the project
- Constantly monitor the situation
- Keep in touch with the stakeholders

>[!example]+
>**Game studio** VS **Internal audit**

## Project closing phase
- Determine what makes a success
- Lessons learned for future application
